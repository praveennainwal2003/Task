A role-based educational platform built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
The application allows **Academies** to upload study PDFs and **Students** to search, preview, and view them securely.

### Frontend
- React.js
- React Router
- Axios
- CSS (custom UI)

### Backend
- Node.js
- Express.js
- MongoDB (Atlas / Local)
- Mongoose
- Multer (PDF uploads)
- JWT (Authentication)

---Backend start 
cd backend
npm install
npm run dev


----frontend start
cd frontend
npm install
npm run dev
